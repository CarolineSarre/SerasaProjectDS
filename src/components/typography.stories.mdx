# Utilização

tipografia sendo utilizada como Tag e recebendo sua variação de cor.

<Source
  language='html'
  dark
  format={false}
  code={dedent`
     <Display classe="color-magenta">
        Texto
    </Display>
  `}
/>

tipografia sendo utilizada e recebendo sua variação de cor via Props.

<Source
  language='js'
  dark
  format={false}
  code={dedent`
     Display.defaultProps = {
  children: null,
  classes: null,
  color: 'dark-high',
};
  `}
/>

**Caso a tipografia receba a cor via classe e via Props, prevalece o que for passado via Props.**
